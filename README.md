## 入室方法

1. オートロックで部屋番号を入力し、呼び出し
2. [@S64](https://github.com/S64)の応答を待つか、呼出がタイムアウトするまでに後述するSlackコマンドで解錠する
3. 部屋前へ移動する
4. 後述するアプリで玄関を解錠するか、インターホン経由で[@S64](https://github.com/S64)の応答を待つ
