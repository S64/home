## 入室方法

1. オートロックで部屋番号を入力し、呼び出し
2. [@S64](https://github.com/S64)の応答を待つか、呼出がタイムアウトするまでに後述するSlackコマンドで解錠する
3. 部屋前へ移動する
4. 後述するアプリで玄関を解錠するか、インターホン経由で[@S64](https://github.com/S64)の応答を待つ

## 権限等

各状態における権限について。

| 操作 | 権限 | 補足 |
|-----|------|-----|
| オートロック解錠 | Slackメンバー, 在室者 | Slackへジョインしていない場合、[@S64](https://github.com/S64)による物理解錠のみ |
| 玄関解錠 / 施錠 | Qrio Lockメンバー, 在室者 | Qrio Lockへ追加されていない場合、[@S64](https://github.com/S64)による物理解錠のみ |
| 居室照明点灯 / 消灯 | Slackメンバー, 在室者 | 音声アシスタント経由で操作可, 調光は[@S64](https://github.com/S64)のみ |
| 浴室, 洗面所照明点灯 / 消灯 | Slackメンバー, 在室者 | 音声アシスタント経由で操作可, 調光は[@S64](https://github.com/S64)のみ |
| エアコン運転 / 停止 | Slackメンバー, 在室者 | 音声アシスタント経由で操作可, モード切替や調整は[@S64](https://github.com/S64)のみ |

### Slackワークスペース

ほとんどの設備操作は音声アシスタント経由で可能ですが、音声を用いずに操作したい場合や、音声アシスタントでの操作が利用できない設備は、Slack経由で操作可能です。  
希望する場合は[@S64](https://github.com/S64)にリクエストしてください。

特に、オートロックの解錠はセキュリティの都合上物理キーを共有できないため、Slack経由での解錠が必要です。

### Qrio Lockアプリ

玄関の解錠はセキュリティの都合上物理キーを共有できないため、Qrio Lock経由での解錠が必要です。  
希望する場合は[@S64](https://github.com/S64)にリクエストしてください。

## 喫煙

室内は禁煙です。また、灰皿等もありません。  
どうしても喫煙したい場合は玄関先、または換気扇の下で喫煙してください。  
ベランダも利用できますが、隣接する住民が被害を被る可能性があります。

または、加熱式たばこや電子たばこであれば少量のみ許容しています。
